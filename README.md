Calculate Numbers
=========

A small library that calculate numbers

## Installation

  `npm install nab-calculate-number`

## Usage

    var calculate = require('nab-calculate-number');

    var result = calculate.sum(1, 2);
  
  
  Output should be `3`


## Tests

  `npm test`
